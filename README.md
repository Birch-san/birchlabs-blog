# birchlabs-blog

New [Jekyll](https://jekyllrb.com/)-powered blog intended for Birchlabs.co.uk

## Development

### Setup computer

#### Get Ruby

Preferably via rvm.

#### Install [Jekyll](https://jekyllrb.com/)

```bash
gem install jekyll
```

### Serve blog

```bash
jekyll serve
# Now browse to http://localhost:4000
```
